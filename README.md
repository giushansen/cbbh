## WEB REQUESTS

### curl options
`curl -k https://inlanefreight.com`  
`-v`: Verbose  
`-vvv`: More verbose  
`-I`: Sends a HEAD request to get and only display reponse headers   
`-i`: Sends any request we specify and prints the headers as well  
`-H`: Set request headers  
`-A`: Set Userger Agent
